## 百度：搬家-切绳子

小明打算搬家，他需要很多绳子。……。总之，小明有n根绳子，他希望对这些绳子进行加工，每根绳子**最多**可以切割一次。他的目的是希望最终可以得到最多根长度相同的绳子，你能帮他计算结果吗？

输入描述

```
第一行一个整数n，表示绳子的数量
第二行n个整数，表示长度
```

样例

```
输入
3
2 3 3
输出
5
解释：5根1.5的
```



[题解](./baidu-001.py)

思路：排序后按照绳子长度逐个考察，去掉切不成1/2长的过短绳，加上相同长度绳子的个数（能切出两份来）。

## boss直聘：分词

给一句话，可以1个字切或2个字切，问有几种切分方式。接收函数输入输出。

样例

```
输入
"欢迎大家"
输出
5
解释：
"欢|迎|大|家"，
"欢|迎|大家"，
"欢|迎大|家"，
"欢迎|大|家"，
"欢迎|大家"
```

[题解](./boss-002.py)

思路：最简单的动态规划，斐波那契数列。
